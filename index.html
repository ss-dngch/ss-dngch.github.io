Things I built

BIOINFORMATICS
GeneScope

Interactive genomics platform for analyzing and visualizing biological datasets. Supports dimensionality reduction (PCA/UMAP), dataset exploration, and pathway-focused insights through a clean web interface. Designed to make complex genomic data more accessible for analysis and interpretation.

GitHub Live Demo
BIOINFORMATICS
Gene Sequence Analyzer

A pipeline for analyzing DNA sequences from public datasets. Uses BLAST alignment and custom Python scripts to identify mutations and annotate genetic variants. Built for a genomics course and expanded as a personal project.

GitHub Demo
ML / DATA
Protein Structure Classifier

CNN-based classifier trained on PDB datasets to predict secondary protein structure from amino acid sequences. Achieved 87% accuracy on test set. First foray into combining deep learning with molecular biology.

GitHub Paper
WEB / APP
Research Paper Tracker

Full-stack web app for organizing and annotating academic papers. Features tagging, search, and a recommendation engine based on reading history. Built with React + Node.js + PostgreSQL. My most used personal tool.

GitHub Live Site
BIOINFORMATICS
Phylogenetic Tree Builder

Interactive visualization tool for building and exploring phylogenetic trees from FASTA input. Uses neighbor-joining algorithm with D3.js for rendering. Helps researchers explore evolutionary relationships visually.

GitHub Try It

Drug Interaction Predictor

Graph neural network model that predicts potential drug-drug interactions using molecular structure data from ChEMBL. Senior capstone project. Explores how ML can assist early-stage pharmacological screening.
